きっかけ

北区に引っ越してから、近くの桜宮・大阪城・桜宮・中之島を散策し始めて、市の中心部にもかかわらず四季折々の自然を楽しめることに驚き !!

やってみたこと

こんなに素晴らしい大阪を広く知ってもらいたい。中之島・大阪城・桜宮・淀川・（願わくば、うめきたまで）をつなぐ緑の回廊（Green Corridor）を四季折々に楽しめるような情報をWebで提供。

joggingに絞り、関係する情報の位置等をまとめてLOD化して、Google map上で見えるようにする。

プログラムとしては、次のような事をjavascriptで行っています。

  • 関連情報の名称、位置等をcsvファイル化
  • csv/RDF変換ツールでRDF化し、LODハッカソンのエンドポイントに登録。ここは、全面的に第5回のLODハッカソンで提供いただいたツールを使用しています。
  • SPARQLのQueryで、名称、位置情報、アイコン番号、リンク情報を検索、取得。ここも、第5回のLODハッカソンで提供いただいたmap表示ツールに検索式とデータ取得を少し追加して使用しています。
  • その情報を、Google Map上に、joggingコース情報と共にアイコンとして設置

できること

ナビゲーションメニューバー:トップとReadme（このページ）間を遷移します。

joggingコース関連情報:トップページに公衆浴場、コインロッカ、公衆便所、水飲み場の位置をマップ。アイコンをリンクすると、外部の関連HPに飛びます。

２月のおすすめ：トップページの右側にカモメと梅林の写真を載せました。それぞれマップ上に大きめのアイコンで位置を示してます。アイコン（並びに、写真）をクリックすると外部の関連HPに飛びます。

動作確認：chrome(40.0.2214.115 m）のみで確認。すみません。

既知の不具合：銭湯が多数表示された事があり、RDFデータ登録をファイル名を変えてやり直しました。その後再現していませんが、念のため。

著作権等：第5回のLODハッカソンで提供いただいた、map表示ツールを使っています。
